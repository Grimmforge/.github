# Grimmforge

**Small, standalone World of Warcraft addons. Each does one thing, properly.**

</div>

---

Grimmforge is where [Grimmvöcare](https://www.curseforge.com/members/grimmvocare)'s
World of Warcraft addons are built and kept. The brief is the same for all of
them: lightweight, focused on a single job the default UI does badly, and
self-contained - no required libraries (unless directly embedded), no telemetry, no RGB.

## In the forge

### Disenqueue

[![Downloads](https://img.shields.io/curseforge/dt/1555478?label=downloads&color=6E40C9&logo=curseforge&logoColor=white)](https://www.curseforge.com/wow/addons/disenqueue)
[![License](https://img.shields.io/badge/license-Apache--2.0-6E40C9)](https://github.com/Grimmforge/Disenqueue/blob/main/LICENSE)

A disenchant queue that processes one item per hardware input - fully within
Blizzard's one-action-per-keypress rules. Scan your bags, bind a key (the
scroll wheel works well), and shred through gear safely. Also handles
Prospecting and Milling, because Enchanting is the one true craft and
everything else is breaking rocks and grinding weeds.

[CurseForge](https://www.curseforge.com/wow/addons/disenqueue) · [Source](https://github.com/Grimmforge/Disenqueue)

### Epithet

[![Downloads](https://img.shields.io/curseforge/dt/1567911?label=downloads&color=6E40C9&logo=curseforge&logoColor=white)](https://www.curseforge.com/wow/addons/epithet)
[![License](https://img.shields.io/badge/license-Apache--2.0-6E40C9)](https://github.com/Grimmforge/Epithet/blob/main/LICENSE)

A quiet hall of names for the one reward WoW never gave a proper home: your
titles. Browse every title in the game, see how each is earned and how rare it
is, filter and search the lot, and equip any of them in two clicks. Built like
a heraldic catalogue rather than a combat overlay.

[CurseForge](https://www.curseforge.com/wow/addons/epithet) · [Source](https://github.com/Grimmforge/Epithet)

## What every Grimmforge addon is

- **Standalone** - no required libraries unless there's a genuine reason for one.
- **Rules-compliant** - nothing that automates input or skirts Blizzard's restrictions.
- **Quiet** - no telemetry, no nags, no phoning home.
- **Open source** - released under the Apache License 2.0.
- **Kept current** - updated each patch as the game changes.
